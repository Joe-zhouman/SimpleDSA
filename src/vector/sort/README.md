# VectorSortTest

## Introduction

An executable to test `Sort Methods`  of `Vector`.

## Installing

```shell
mkdir build
cd build
cmake ..
make
```

## Usage

```shell
./VecterSortTest -[p] [parameter]
./VecterSortTest --[option] [parameter]
```

parameter should be a positive integer.

## Examples

### bubule sort

`INPUT`

```shell
./VecterSortTest -b 10
```

---

or

```shell
./VecterSortTest --bubble 10
```
