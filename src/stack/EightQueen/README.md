# EightQueen

## Introduction

An executable to solve `eight queens problem` with a size of `N`.

## Installing

```shell
mkdir build
cd build
cmake ..
make
```

## Usage

```shell
./EightQueens [n]
```

n should be a positive integer.

`EXAMPLE`

---

### input

```shell
./EightQueens 4
```
